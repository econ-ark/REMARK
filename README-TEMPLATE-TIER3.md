## Parameter Modification Guide

### Main Parameters

Edit `code/config.py` to modify key parameters:

```python
# Example parameters
SAMPLE_SIZE = 10000        # Number of simulations
TIME_PERIODS = 100         # Periods to simulate
DISCOUNT_FACTOR = 0.96     # Household discount factor
```

### Running with Modified Parameters

```bash
# Edit parameters
nano code/config.py

# Run with new parameters
./reproduce.sh
```

### Parameter Sensitivity

Key parameters affecting results:
- `DISCOUNT_FACTOR`: Affects consumption dynamics (see Table 2)
- `SAMPLE_SIZE`: Affects precision but not qualitative results
- `TIME_PERIODS`: Affects convergence (≥100 recommended)

### Common Issues

**Solution**: Update conda (`conda update conda`) and retry

**Problem**: Numerical results differ slightly  
**Solution**: Minor differences (<0.01%) expected across platforms due to floating-point arithmetic. Qualitative results should be identical.

**Problem**: Code runs but produces no output  
**Solution**: Check `reproduce.log` for error messages
